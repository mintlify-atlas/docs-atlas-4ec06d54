# /usr/sbin utilities

The `/usr/sbin` directory contains **system administration tools** that are not required for single-user mode boot but are needed for a fully running system. These tools configure services, manage users, maintain the network, and perform system maintenance. Most require root privileges.

The source lives in `usr.sbin/` in the MidnightBSD source tree, containing over 240 subdirectories.

## Key utilities in detail

### sshd — OpenSSH server

```bash
sshd -t                      # Test configuration
sshd -d                      # Debug mode (single connection)
sshd -f /etc/ssh/sshd_config # Alternate config
service sshd start
service sshd restart
service sshd status
```

Key configuration options in `/etc/ssh/sshd_config`:

```text
Port 22
PermitRootLogin no
PasswordAuthentication yes
PubkeyAuthentication yes
AuthorizedKeysFile .ssh/authorized_keys
X11Forwarding no
AllowUsers alice bob
```

### cron — periodic task scheduler

System crontabs are in `/etc/crontab` and `/etc/cron.d/`. Per-user crontabs are managed with `crontab(1)`.

```bash
crontab -e           # Edit current user's crontab
crontab -l           # List current user's crontab
crontab -r           # Remove current user's crontab
crontab -u alice -l  # View another user's crontab (root)
```

Crontab format:
```text
# min  hour  mday  month  wday  command
  0    2     *     *      *     /usr/bin/find /tmp -mtime +7 -delete
  */5  *     *     *      *     /usr/local/bin/check-service
```

### ntpd — Network Time Protocol daemon

```bash
service ntpd start
ntpq -p              # Show peer status
ntpq -c associations # Show associations
```

Configuration in `/etc/ntp.conf`:

```text
pool 0.midnightbsd.pool.ntp.org iburst
pool 1.midnightbsd.pool.ntp.org iburst
driftfile /var/db/ntp.drift
leapfile /var/db/ntpLeapSeconds
```

### named / unbound — DNS

MidnightBSD ships **Unbound** as the local DNS resolver and caching nameserver.

```bash
service local_unbound start
unbound-checkconf            # Validate config
unbound-control reload       # Reload without restart
unbound-control dump_cache   # Show cache
unbound-control flush example.com  # Flush specific entry
```

Configuration: `/var/unbound/unbound.conf`

### sendmail

MidnightBSD includes Sendmail as the default MTA. The `sendmail` command in `/usr/sbin/` is the Postfix-compatible wrapper.

```bash
service sendmail start
sendmail -bt                  # Address test mode
sendmail -bv user@example.com # Verify routing
mailq                         # Show mail queue
newaliases                    # Rebuild /etc/aliases database
```

### ppp — user-space PPP daemon

```bash
ppp -background provider      # Connect and background
ppp -direct ttyUSB0           # Direct serial PPP
ppp -auto provider            # Dial on demand
pppctl                        # Interact with running ppp
```

### wpa_supplicant — wireless authentication

`/usr/sbin/wpa/wpa_supplicant` handles WPA/WPA2/WPA3 authentication for wireless interfaces.

```bash
wpa_supplicant -i wlan0 -c /etc/wpa_supplicant.conf -B
wpa_cli status
wpa_cli scan
wpa_cli scan_results
```

Example `/etc/wpa_supplicant.conf`:

```text
ctrl_interface=/var/run/wpa_supplicant
network={
    ssid="myhomenet"
    psk="mysecretpassphrase"
    key_mgmt=WPA-PSK
}
```

### periodic — scheduled maintenance

`periodic(8)` runs daily, weekly, and monthly maintenance scripts from `/etc/periodic/` and `/usr/local/etc/periodic/`. It is invoked by the system crontab.

```bash
periodic daily               # Run daily maintenance now
periodic weekly
periodic monthly
```

To enable/disable individual jobs, edit `/etc/periodic.conf`:

```bash
# /etc/periodic.conf
daily_clean_tmps_enable="YES"
daily_status_disks_enable="YES"
weekly_locate_enable="YES"
monthly_accounting_enable="YES"
```

### service — rc.d service control

```bash
service sshd start
service sshd stop
service sshd restart
service sshd reload
service sshd status
service -e                  # List enabled services
service -l                  # List all known services
```

### sysrc — rc.conf management

```bash
sysrc sshd_enable="YES"         # Enable SSH server
sysrc sshd_enable               # Query value
sysrc -x sshd_flags             # Remove a variable
sysrc -f /etc/rc.conf.local ntpd_enable="YES"  # Specific file
```

### jail — OS-level virtualization

```bash
jail -c path=/jails/myjail host.hostname=myjail.local \
     ip4.addr=10.0.0.10 command=/bin/sh
jls                             # List running jails
jexec myjail csh                # Execute shell in jail
jail -r myjail                  # Remove jail
```

Persistent jails are configured in `/etc/jail.conf`.

### midnightbsd-update — binary base updates

```bash
midnightbsd-update fetch        # Download updates
midnightbsd-update install      # Apply updates
midnightbsd-update rollback     # Revert last update
```
